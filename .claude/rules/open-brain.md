# Open Brain — Shared Semantic Memory

Cross-repo memory via `open-brain` MCP. Use `brain_search` before complex tasks; `brain_remember` after solving non-trivial problems or discovering gotchas.

- **Always** set `source` to `pushflip` for all queries and writes in this repo. Do not mix memories with other projects.
- When searching, filter by `source: pushflip` to avoid pulling in unrelated context from other repos.
- Categories: `decision`, `debug`, `gotcha`, `general`.
- Write concise content (1-3 sentences). Include the "why", not just the "what".
